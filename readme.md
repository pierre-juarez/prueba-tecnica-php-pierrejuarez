
# Prueba Técnica PHP

Pequeña prueba técnica sobre PHP y POO, así como el uso de Test con PHPUnit.

## Running Tests

Para correr los test, verifique que tenga composer instalado, y digite el siguiente comando en su consola.

```bash
  vendor/bin/phpunit
```
